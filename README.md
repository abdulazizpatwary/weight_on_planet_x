# Weight on Planet X

A simple **Flutter app** that calculates your weight on different planets based on your Earth weight. This project demonstrates the use of **TextField**, **Radio Buttons**, **Stateful Widgets**, and basic **UI layout** in Flutter.

## Features

- Enter your weight on Earth.
- Select a planet: **Pluto**, **Mars**, or **Venus**.
- Calculate your weight on the selected planet.
- Clean and simple user interface.
